# OncoCorr

**AI-Augmented Multi-Omics Research & Gene Correlation Dashboard**

OncoCorr is a React + TypeScript dashboard that integrates genomics, transcriptomics, proteomics, and epigenomics data to surface gene correlations and AI-driven biomarker insights.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. The project is configured for deployment on Vercel.
